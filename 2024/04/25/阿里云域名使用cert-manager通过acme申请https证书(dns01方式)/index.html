    阿里云域名使用cert-manager通过acme申请https证书(dns01方式)

    前提条件

    • k8s集群<1.19
    • cert-manager
    • ingress-nginx
    • 阿里云域名

    申请AccessKey

    登录阿里云域名所在账号，开通AccessKey并记录下来

    将access-key和secret-key转换成base64

    将转换后的值保存为k8s secret

    安装alidns-webhook

    启动一个http服务

    测试用，如果有服务可跳过

    颁发机构（CA）

    签发证书

    等待certificate状态为True

    测试

    浏览器通过https访问域名

    故障排查

    Certificate => CertificateRequest => Order => Challenge

    以此使用kubectl get、describe查看状态和日志
