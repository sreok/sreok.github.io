    • 【虚拟化】p2p：Ghost备份恢复windows物理机

    备份windows

    1. 准备PE盘插入电脑，开机从U盘启动

    2. 通过ghost备份磁盘：依次点击Local -> Partition -> To Image

    3. 选择磁盘

    4. 选择分区

    5. 选择保存位置

    6. 导出格式为GHO

    还原主机

    1. U盘启动进入pe系统

    2. 磁盘分区工具分区C盘MBR格式

    3. 打开Ghost工具依次点击：Local -> Partition -> From Image

    4. 选择备份的GHO文件

    5. 选择还原到C盘，开始还原

    已知问题

    1. not enough room on target partition（目标分区容量不够）

    2. An internal inconsistency has been detected（检测到内部不一致）

      1. 排查思路：

      2. 更换分区格式NTFS为FAT32，与恢复前的分区格式对应

      3. 更换PE为OneKey Ghost，雨林木风出版的（未验证）

      4. 使用GhostExplorer打开GHO文件，查看创建此GHO文件的GHOST的版本，使用相同版本

      5. 使用GhostExplorer提取出文件，做成ISO镜像，安装到C盘
